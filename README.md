## ☆ Projeto

O projeto tem o obetivo de criar uma página da web que simula uma Pokedex, um dispositivo eletrônico usado para armazenar informações sobre as várias espécies de Pokémon. O primeiro código é o HTML, que define a estrutura da página, incluindo elementos como imagens, formulários e botões. O segundo código é o CSS, que define o estilo visual da página, incluindo fontes, cores, margens e posicionamento de elementos. O terceiro código é o JavaScript que junto de uma aplicação web que permite ao usuário pesquisar e visualizar informações sobre Pokémons ([PokeAPI](https://pokeapi.co)), e adiciona interatividade à página permitindo que os usuários pesquisem Pokémon por nome ou número e naveguem pelos resultados usando os botões "Prev" e "Next". Em conjunto, esses códigos criam uma página interativa e estilizada que permite aos usuários explorar informações sobre as mais atualizadas gerações de Pokémon.
